Proving Syllogisms

There are 8 buttons in the present system. Their use can be explained in the following manner:

New

Picks a new syllogism at random.

Trans

Will - if possible - prove new propositions using the following deduction:

All Y are Z

All X are Y

Therefore: All X are Z

Subst

Will - if possible - prove new propositions using the following deduction:

All Y are Z

Some X are Y

Therefore: Some X are Z

Contra

Will - if possible - prove new propositions using the following deduction:

All X are Y

Therefore: All non-Y are non-X

(Here, non-X corresponds to the set of elements not belonging to X.)

Mut

Will - if possible - prove new propositions using the following deduction:

Some X are Y

Therefore: Some Y are X

Ex

Will - if possible - prove new propositions using the following deduction:

All X are Y

Therefore:Some X are Y

  

Buttons for final decision

When you have explored the provability of the syllogism in question, it should be evaluated.

Use the button...

  • Invalid, if you after your explorarion find that the syllogism in question is invalid.
  • Valid,  if you think that the syllogism in question is valid.
