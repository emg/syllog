The Logic of Syllogisms


The classical syllogisms may be proved using the following rules. The procedure can be tested below.

Subst: A universal proposition may allow a substitution in an existential proposition.
            [All: x] (IF: ([x] -attr-> [A]) THEN: ([x] -attr-> [B])             [All: x] (IF: G1 THEN: G2)
            [x] ( ([x] -attr-> [A])  ([x] -attr-> [C]) )                                [x] (G1 G3)
            Ergo:                                                                                    Ergo:
            [x] ( ([x] -attr-> [B])  ([x] -attr-> [C]) )                                [x] (G2 G3)

Contra: A universal proposition may be transformed into another universal proposition by contraposition.
            [All: x] (IF: ([x] -attr-> [A]) THEN: ([x] -attr-> [B]))            [All: x] (IF: G1 THEN: G2)
            Ergo:                                                                                     Ergo:
            [All: x] ~(IF: ([x] -attr-> [B]) THEN: ~([x] -attr-> [A])         [All: x] (IF: ~G2 THEN: ~G1) 

Trans: Two universal propositions may be combined by transition.
            [All: x] (IF: ([x] -attr-> [A]) THEN: ([x] -attr-> [B])             [All: x] (IF: G1 THEN: G2)
            [All: x] (IF: ([x] -attr-> [B]) THEN: ([x] -attr-> [C])             [All: x] (IF: G2 THEN: G3)
            Ergo:                                                                                    Ergo:
            [All: x] (IF: ([x] -attr-> [A]) THEN: ([x] -attr-> [C])             [All: x] (IF: G1 THEN: G3)

Exist: An existential proposition follows from the corresponding universal proposition.
            [All: x] (IF: ([x] -attr-> [A]) THEN: ([x] -attr-> [B])            [All: x] (IF: G1 THEN: G2) 
            Ergo:                                                                                   Ergo:
            [x] ( ([x] -attr-> [A])  ([x] -attr-> [B]))                                [x] (G1 G2)
